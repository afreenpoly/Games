# To-Do List App

This is a simple to-do list application built with Node.js and the Express framework. The application allows the user to create a list of tasks that they need to complete. 

## Installation

1. Clone the repository to your local machine.
2. Install the dependencies by running `npm install` in the project directory.

## Usage

1. Start the application by running `node app.js` in the project directory.
2. Open your web browser and navigate to `http://localhost:3000`.
3. You should see the application interface, which allows you to add tasks to the to-do list.

## Features

- Add new tasks to the to-do list by entering text in the input field and clicking the "+" button.
- Mark tasks as completed by checking the checkbox next to the task.
- View the current date, which is displayed at the top of the page.
- The application is built using the EJS templating language, which allows for easy customization of the user interface.

## Dependencies

- Express: A popular Node.js framework for building web applications.
- body-parser: A middleware that allows the application to parse incoming request bodies.
- EJS: A templating language for generating HTML pages.
